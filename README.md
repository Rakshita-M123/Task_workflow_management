# TaskFlow Task Tracker

## Overview

TaskFlow is a workflow and task management dashboard developed using Python, Pandas, and Streamlit. The application helps users track project tasks, monitor progress, and manage workflow activities efficiently.

## Features

- Task tracking and monitoring
- Project progress visualization
- Status-based reporting
- Interactive dashboard
- Workflow management

## Technologies Used

- Python
- Pandas
- Streamlit

## Project Structure

TaskFlow-Task-Tracker
│
├── app.py
├── tasks.csv
├── README.md
└── requirements.txt

## Installation

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
